ApiTab

Local-First API Testing Tool | Type: Browser Extension + Desktop App | Ongoing solo project

A fast, minimal, local-first alternative to Postman. Ships as a browser extension (Chrome/Firefox) and a native Electron desktop app sharing one React/TypeScript codebase, with an optional Laravel backend for team-shared collections and variables.

Skills Utilized:

TypeScript, React, WXT, Electron, Zustand, Tailwind CSS, PHP, Laravel, Sanctum, SQLite, GitHub Actions CI/CD.

  1. Request builder, environments with variable interpolation, and a Postman-script-compatible pre-request/post-response sandbox.
  2. Imports Postman Collection and Environment exports directly.
  3. Optional team workspaces with role-based collection sharing and a per-team shared-variable pool.
