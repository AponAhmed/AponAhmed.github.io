Bulk Mailer Desktop App

Bulk Email Sender via Windows / Mac

Duration: 2 Months | Deployed single-handedly

Overview

Java bulk mailing application seamlessly integrates with the Gmail API, facilitating automated email sending. Supporting multiple Google accounts and SMTP accounts, it offers flexibility for users managing diverse accounts concurrently. As a desktop application, it ensures local accessibility and convenience for users.

The inclusion of an HTML Editor, utilizing JEditorPane and StyledEditorKit, empowers users to compose and format HTML content for their bulk emails within the application. The user-friendly interface, coupled with robust error handling and logging, guarantees a smooth user experience and provides insights into application performance.

Key Features

Gmail API Integration

Seamless integration with Gmail API for automated email sending with authentication and authorization

Multiple Account Support

Manage multiple Google accounts and SMTP accounts concurrently for diverse email campaigns

HTML Email Editor

Built-in HTML editor using JEditorPane and StyledEditorKit for rich email composition

Database Integration

SQLite for local storage and MySQL for remote with auto-synchronize data capabilities
